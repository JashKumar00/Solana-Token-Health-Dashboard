=== WHAT THE APP DOES ===

User types a Solana token name, symbol, or mint address into a search bar.
App calls Jupiter APIs and renders a complete token health dashboard in real time.

=== FEATURE 1: Token Search (Tokens API) ===

Endpoint: GET https://api.jup.ag/tokens/v1/search?query=SEARCH_TERM
- As user types (debounced 500ms), call the search endpoint
- Show a live dropdown of matching tokens with: logo, name, symbol
- User clicks a result → load the full dashboard for that token
- Also accept a raw mint address pasted directly into the search bar
- All results come from Jupiter API — no hardcoded token list

=== FEATURE 2: Token Metadata Card (Tokens API) ===

Endpoint: GET https://api.jup.ag/tokens/v1/token/MINT_ADDRESS
Fetch and display:
- Token name (real, from API)
- Token symbol (real, from API)
- Token logo image (real URL from API, shown in glowing circle)
- Mint address (truncated: first 4 chars...last 4 chars, click to copy full)
- Verification status (verified true/false from API)
- Decimals
- Tags (if any returned by API)
- Daily volume (from API if available)
- Created date (if available)

Display as a hero card at the top of the dashboard.

=== FEATURE 3: Live Price Card (Price API) ===

Endpoint: GET https://api.jup.ag/price/v2?ids=MINT_ADDRESS
Fetch and display:
- Current USD price (real, from API) — large display, animated counter
- Price shown with full decimal precision from API
- Confidence level (if returned by API)
- Last updated timestamp
- Auto-refresh every 30 seconds
- Show countdown bar: "Refreshing in 28s..." that depletes visually
- On each refresh, compare to previous price:
  - If higher: flash green, show ↑ X% change
  - If lower: flash red, show ↓ X% change
  - If same: show — 0.00%
- If price fetch fails: show "Price Unavailable: [exact API error]"

=== FEATURE 4: Organic Score & Trust Signals ===

From the Tokens API response, extract and display:
- Organic score (number from API — display as a score out of 100 with 
  an animated circular progress ring)
- Score color: 
  - 70-100: Solana green #14F195 (HEALTHY)
  - 40-69: yellow #FFB800 (CAUTION)  
  - 0-39: red #FF4444 (RISKY)
- Verification status shown as large badge
- Show a "Token Health Rating" label:
  - Verified + organic score > 70 = "✓ SAFE TO MONITOR"
  - Verified + score 40-70 = "⚠ MODERATE RISK"
  - Unverified OR score < 40 = "✗ HIGH RISK — Trade with caution"
- All values come from real API response only

=== FEATURE 5: Trading Metrics Panel ===

From the Tokens API response, display whatever trading metrics 
the API returns, such as:
- 24h volume (if available in API response)
- Market cap (if available)
- Liquidity (if available)
- Number of holders (if available)
- Any other trading_metrics fields returned by the API

Display each metric in its own small glowing stat card.
If a field is not returned by the API, show "N/A — not provided by API" 
Do NOT make up or estimate any value.

=== FEATURE 6: Raw API Response Viewer ===

At the bottom of the page, add a collapsible section called 
"Raw API Data (for developers)"
- Shows the exact JSON response from both API calls
- Syntax highlighted (use a simple JS highlighter or pre tag with color)
- This helps you document the real API behavior for your DX report

=== FEATURE 7: Search History ===

- Store the last 5 searched tokens in localStorage
- Show them as quick-access chips below the search bar
- Each chip shows token logo + symbol
- Clicking re-loads that token's dashboard instantly

=== LOADING STATES ===
- When search is triggered: show a skeleton loader (pulsing gray blocks)
  in the shape of each card — NOT a spinner, NOT "loading..."
- Skeleton cards should match the exact dimensions of real cards
- Once data arrives: fade in real content replacing skeletons
- Never show empty cards with no explanation

=== ERROR STATES ===
- API key invalid: "Jupiter API returned 401 — check your API key"
- Token not found: "No token found for this address on Jupiter"
- Rate limited: "Jupiter API rate limit hit — try again in a few seconds"
- Network error: show exact fetch error message
- Never show a blank screen — always explain what went wrong

=== BACKEND ROUTES ===
GET /api/token/:mintAddress     → fetch token metadata from Tokens API
GET /api/price/:mintAddress     → fetch live price from Price API
GET /api/search?query=TEXT      → search tokens from Tokens API
GET /                           → serve the frontend HTML file

All routes: wrap in try/catch, return real error from Jupiter on failure.
