Build a Jupiter Token Health Dashboard — a fully animated 3D crypto dashboard. Single HTML/CSS/JS app, no frameworks.
SETUP: index.html + style.css + main.js + globe.js + particles.js + gauge.js. Use Three.js r128 from CDN (https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js). Google Fonts: Orbitron + Share Tech Mono.
BACKGROUND: Canvas particle star field (200 particles, mouse parallax). Three.js rotating purple wireframe sphere (Solana globe) behind hero. CSS aurora light beams (purple+cyan, slow 8s keyframe loop).
NAV: Glassmorphism navbar (backdrop-filter blur 20px). Animated rotating gem SVG logo. "JUPITER TOKEN HEALTH" in Orbitron font. LIVE green dot badge. Scrolling token ticker marquee above nav — green neon glow for gains, red pulse for losses. Tokens: WIF, POPCAT, TRUMP, PEPE, DOGE, SOL, BONK, JUP.
HERO: Giant "TOKEN HEALTH DASHBOARD" title in Orbitron 900 weight with animated gradient text (purple→cyan→pink, background-size 300%). CSS text-shadow glow layers in purple and cyan. 3 orbiting CSS coin sprites (SOL = purple circle, JUP = cyan diamond, BONK = amber hexagon) each with @keyframes orbit + rotateY flip animation. Subtitle: "Real-time Solana token intelligence — cut through the noise." Eyebrow text: "SOLANA · DEFI · MEME INTELLIGENCE" in small spaced caps, cyan color.
SEARCH BAR: Centered glassmorphism search bar, max-width 600px. On focus: rotating conic-gradient border ring animation pulses purple→cyan→pink. Placeholder: "Search token name, symbol, or paste mint/wallet address..." Right side: "SCAN" button with gradient background + hover glow. Recent chip badges below: Wallet · Fartcoin · Punch.
ORGANIC SCORE: Canvas 2D arc gauge. Dark purple background arc, foreground arc with gradient stroke purple→cyan. Center shows score number (78) in Orbitron with count-up animation on page load. Tick marks every 10 units around ring. Score tier pills: 90+ EXCELLENT (cyan) · 70–89 GOOD (blue) · 50–69 MIXED (amber) · below 50 CAUTION (red).
FEATURE CARDS: 4-column glassmorphism grid — Price Chart, Market Stats, Wallet View, Raw API Data. Each card uses backdrop-filter blur with dark background and color accent top border. On hover: JS mousemove drives CSS transform perspective(800px) rotateX() rotateY() for 3D tilt effect. Neon box-shadow glow on hover matching each card's accent color. Animated SVG icon per card.
POWERED BY SECTION: 3 glowing source cards — JUPITER API · TRADINGVIEW · ON-CHAIN DATA. Each with glowing border, icon, short description, subtle scan-line animation overlay.
CTA / FOOTER: Full-width section with aurora background. Big Orbitron text: "Know what's real. Trade smarter." Three glowing pill badges: FREE · NO LOGIN · REAL-TIME. Footer: "Data by Jupiter · DexScreener · GeckoTerminal" in muted small text.
COLOR PALETTE: #020008 background · #7c5fff purple · #00f5d4 cyan · #ff2d78 pink · #ffb86c amber. ALL UI elements use glassmorphism (backdrop-filter blur). No solid opaque backgrounds except the page itself.
TECHNICAL RULES: Pure HTML/CSS/JS only — no React, no Vue, no jQuery. Fully responsive — stack to 1 column on mobile below 768px. Use requestAnimationFrame for all canvas animations. Three.js loaded from CDN only. Entry point is index.html, runs on Replit static server.
